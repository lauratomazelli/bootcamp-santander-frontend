Entendendo HTML

O que é HTML?

HTML significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto. É a base de qualquer página da web. Com ele, podemos estruturar conteúdo, como textos, imagens e vídeos.

Por que ele é tão importante?

Porque sem HTML, não existiria a internet como conhecemos! É ele que define o que é um título, um parágrafo, uma lista, um link...

Principais Tags

Listas ordenadas:

  1. <h1> até <h6> - Títulos
  2. <p> - Parágrafos
  3. <strong>, <i>, <u> - Destaques

Listas não ordenadas:

  • <mark> - Marca texto
  • <small> - Texto pequeno
  • <blockquote> - Citação
